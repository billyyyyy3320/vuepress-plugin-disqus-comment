# vuepress-plugin-Disqus

VuePress plugin for Disqus. Based on [vue-disqus](https://github.com/ktquez/vue-disqus).

## Install

```bash
yarn add vuepress-plugin-Disqus -D
# OR npm install vuepress-plugin-Disqus -D
```

## Usage

```javascript
// .vuepress/config.js
module.exports = {
  plugins: [
    "vuepress-plugin-Disqus",
    {
      /* options */
    }
  ]
};
```
